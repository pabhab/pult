CSV link iz Google Sheets
Kako dobiti CSV link:
1. Otvori tabelu u Google Sheets
2. File → Share → Publish to web
3. Levo odaberi PULLT (naziv svog lista)
4. Desno odaberi Comma-separated values (.csv)
5. Klikni Publish → kopiraj link
6. Nalepi link gore i klikni Učitaj
